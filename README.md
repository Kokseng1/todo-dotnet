Basic crud operation backend using dotnet and SqlServer

1. Authentication

    User authentication is implemented using JWT (JSON Web Tokens).
    Secure login and registration endpoints.
    Role-based access control (optional).

2. CRUD Operations

    Full CRUD (Create, Read, Update, Delete) operations for managing tasks.

3. Pagination

    Paginated retrieval of tasks to efficiently handle large datasets.

4. Filtering and ordering

    Filter tasks based on:
        Task name
    Order based on:
        Created data, task name

Prerequisites

    .NET SDK
    SQL Server
